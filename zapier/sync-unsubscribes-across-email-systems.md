## 📝 What it does
Propagates an unsubscribe in one tool to all other email systems for compliance.

## 🎯 Use when
> When you run multiple email platforms.

## ⚡ Trigger
New unsubscribe

## 🏁 Steps
1. Trigger on unsubscribe
2. Find contact in each system
3. Mark unsubscribed everywhere
4. Log change

## ⚠️ Note
Critical for CAN-SPAM/GDPR consistency.
