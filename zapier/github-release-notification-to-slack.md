## 📝 What it does
Auto-detects new releases in a specific GitHub repository and posts a formatted message with release notes directly to a Slack channel.

## 🎯 Use when
> To keep development and product teams informed of software releases in real time.

## ⚡ Trigger
New Release in GitHub Repository

## 🏁 Steps
1. Monitor GitHub repo for a new release event.
2. Extract release name, tag, and description changelog.
3. Send message using Slack Integration to specified channel.

## ⚠️ Note
Set up webhook events on GitHub for clean webhook response delivery.
