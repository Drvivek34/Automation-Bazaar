## 📝 What it does
Parses structured emails (orders, alerts) and writes records into Airtable.

## 🎯 Use when
> When inbound emails carry data you need in a table.

## ⚡ Trigger
New email matching a filter

## 🏁 Steps
1. Watch mailbox
2. Parse fields (text parser)
3. Map to Airtable schema
4. Create/Update record

## ⚠️ Note
Build the parser against several real samples.
