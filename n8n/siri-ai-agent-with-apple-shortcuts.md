## 📝 What it does
Bridges iOS Siri voice shortcuts to a powerful agentic backend in n8n, enabling deep custom commands via voice.

## 🎯 Use when
> For voice-activated smart home commands or hands-free task triggers on iPhones.

## ⚡ Trigger
iOS Apple Shortcut webhook call

## 🏁 Steps
1. Receive Siri shortcut webhook payload.
2. Parse voice input transcripts.
3. Call LLM Agent with tool calling to process request.
4. Return custom speech response text to Siri for voice synthesis.

## ⚠️ Note
Requires setting up an Apple Shortcut to capture audio and trigger the webhook.
