## 📝 What it does
Turns one blog post into platform-tailored LinkedIn posts, tweets, and an email newsletter draft.

## 🎯 Use when
> When you want to repurpose long-form content into many channels.

## ⚡ Trigger
New blog post (RSS/CMS)

## 🏁 Steps
1. Fetch the new post
2. Summarize + rewrite per platform with an LLM
3. Generate variants
4. Queue/schedule to each channel

## ⚠️ Note
Add brand-voice guidelines to the prompt for consistency.
