## 📝 What it does
Monitors RSS feeds for new tech articles, extracts key points, generates engaging tweets using OpenAI GPT models, and auto-posts them to Twitter/X.

## 🎯 Use when
> When you want to maintain a consistent social media presence without writing daily content manual drafts.

## ⚡ Trigger
RSS Feed Trigger

## 🏁 Steps
1. Trigger when a new item is published to RSS feed.
2. Parse article content and summarize using OpenAI node.
3. Formulate punchy tweet with hashtags.
4. Post to Twitter/X using the Twitter node.

## ⚠️ Note
Requires RSS URL, OpenAI API Key, and Twitter OAuth credentials.
