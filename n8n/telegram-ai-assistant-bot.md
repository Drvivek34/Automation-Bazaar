## 📝 What it does
A Telegram bot backed by an LLM with memory, for Q&A and task commands.

## 🎯 Use when
> When you want a personal assistant in Telegram.

## ⚡ Trigger
Telegram message

## 🏁 Steps
1. Receive Telegram update
2. Load short-term memory
3. Call the LLM
4. Reply in chat + persist memory

## ⚠️ Note
Scope the bot to allowed chat IDs to prevent abuse.
