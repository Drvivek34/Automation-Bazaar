## 📝 What it does
Scheduled dependency upgrades that open a PR and run CI before merge.

## 🎯 Use when
> When you want safe, steady dependency hygiene.

## ⚡ Trigger
Schedule (weekly)

## 🏁 Steps
1. Scan manifests for updates
2. Open grouped PRs
3. Run CI
4. Auto-merge if green

## ⚠️ Note
Group minor/patch updates to reduce PR noise.
